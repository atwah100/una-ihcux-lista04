O try-catch é uma estrutura usada na programação para lidar com erros que podem acontecer durante a execução do código.
No bloco “try”, colocamos o código que pode gerar algum erro. Já no bloco “catch”, definimos o que o programa deve fazer caso esse erro aconteça.
Ele não impede que o erro ocorra, mas evita que o programa pare de funcionar de forma inesperada. 
Em vez de travar, o sistema consegue tratar o problema e continuar executando ou mostrar uma mensagem mais amigável para o usuário.
Assim, o try-catch está diretamente ligado à prevenção de erros porque ajuda a controlar falhas, tornando o sistema mais estável, 
seguro e com uma melhor experiência para quem está utilizando.
